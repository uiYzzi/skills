# 词级清单

> 分三级。常现即禁是 AI 文的标志，单独出现就该处理。聚类才禁单独可放行，同段两个以上是信号。密度过高才禁是常见词，只在全文密度明显过高时处理。

## 常现即禁

### 解释与提示性短语

删掉提示层，直接给内容。

- 可以理解为、本质上、具体要看、取决于、一般来说、通常、严格来说
- 先说结论、一句话总结、简单的说、说人话就是、结论先说
- 不绕弯子、归根结底、你可以把它看成、如果你需要我可以
- 值得注意的是、值得一提的是、不难发现、显而易见、毫无疑问、不可否认
- 毋庸置疑、不可否认、从某种程度上、在一定程度上、事实上

### 总结收尾词

- 综上所述、总的来说、总之、总而言之、最终来看、说到最后、简而言之
- 归根结底、一句话概括、最终还是要回到

### 套话开场

- 在这个时代、随着技术发展、众所周知、近年来
- 让我们一起、让我们来看看、接下来让我们、本文将、下面我会

### 协作交流痕迹

- 希望这对你有帮助、当然、一定、您说得完全正确、您想要、请告诉我
- 截至我的知识、根据我最后的训练更新、作为 AI、我无法访问最新信息

### 谄媚语气

- 很高兴、非常荣幸、感谢你的提问、令人振奋、期待与您一起
- 好问题、好观点、你问到了问题的核心、顶刊作者的素养

## 聚类才禁

单独出现可放行，同段两个以上是 AI 信号。

### 高频连接词

- 此外、然而、因此、换句话、与此同时、另一方面
- 由此可见、由此、从而、进而、进一步、在此基础上
- 事实上、实际上、相对来说、相对而言

### 渲染性修饰词

- 深入、深度、全面、系统、核心、关键、重要
- 显著、卓越、革命性、颠覆性、开创性、突破性
- 无缝、直观、强大、稳健、富有洞见

### 调试腔与姿态词

- 根因、收口、落盘、坐实、对上了、锁住、更硬
- 兜住、接住、收敛、坍缩、稳稳接住

### 暴力动作腔

- 砍一刀、补一刀、钉死、狠狠干、拍脑门、扒开、揪出来、捞出来

### 伪口语化动作词

- 拆一拆、盘一盘、捋一捋、聊一聊、划重点、敲黑板、掰开来看
- 拆完、理完、看完、弄明白（步骤承接，逐词黑名单抓不住，从句式层拦截）

### 高频分析动词

- 拆解、梳理、剖析、解构、聚焦、洞察、深耕、赋能、助力、践行、驱动、构建、打造
- 拆解、梳理、剖析是 AI 中文分析类文本的签名词

### 降维解释引导语

- 说白了、本质上、归根结底、一句话概括、简单来说、换个角度看、从某种意义上说、不妨这样理解

### 伪坦率评价词

- 直白、坦率、坦白、赤裸、赤裸裸、无遮拦、明白无误、不加掩饰、不容回避
- 典型句式：X 说得很直白、答案直白得多、数据说得更直白。删评价标签，让事实自己说话。

### 伪严谨评价词

- 严谨、严密、严苛、严格、精准、审慎、克制、沉稳、从容
- 典型句式：严谨地探讨、严密的论证、严苛的标准。挂在可核对对象上才保留，空挂的一律删。

## 密度过高才禁

常见词，只在全文密度明显过高时处理，换一部分成具体例子或数字。

- 重要、关键、核心、提升、增强、优化、改善
- significant、innovative、effective、dynamic、scalable、compelling
- 成功、有效、显著、明显

## AI 高频词专表

### 伪学术腔

在论文里有精确含义，AI 当万能标签塞进散文。命中即改。

- 舆论场、话语场、权力场、公共场域、意义场域（场和场域后缀批量造抽象名词）
- 宏大叙事、国家叙事、官方叙事、主流叙事、元叙事（叙事当说法的高级同义词滥用）
- 摊开来看、摊开来说（假装口语化，实则在跑模板）
- 底层逻辑、顶层设计、认知升级、降维打击、赛道、红利、抓手、闭环、存量、增量
- 范式、路径依赖（非经济学语境）、权柄、话语权（非学术引用）

改法：删术语壳，换具体名词。舆论场改几家媒体几个平台某类账号，官方叙事改官方说法，摊开来看直接给判断。

### 政论媒体术语

新闻学宣传学里有精确含义，AI 当高级同义词滥用。

- 口径、出口径、统一口径、对齐口径、通稿口径、官方口径
- 定调、定过调、给 X 定了调、定调子、调子定下来、调门、起调
- 话术、这一套话术、那一套话术、一整套话术
- 通稿、通稿写法、像通稿、念通稿、喉舌（非历史语境）

改法：口径改说法或原话，定调改先表态或带头说，话术改说辞，通稿改统一文本或官方文本。配套警惕 X 研究部和 Y 咨询部这类机构串联做权威铺垫，要么给具体原话，要么删一个机构。

### 戏剧化揭露修辞

- 遮羞布、最后一块遮羞布、面具、画皮、伪装、外衣、皇帝的新衣、幌子、烟幕弹
- 扯下、撕下、揭下、摘下、掀开配遮蔽物
- 剥开、戳穿、戳破、揭穿、撕开、揭开配表象真面目本质真相外壳包装

改法：删戏剧化动作隐喻，直接给事实和判断。

### 极值判断词

- 最残酷的地方在于、最可怕的是、真正可怕的是、讽刺之处在于、更残酷的是
- 吊诡之处在于、可怕之处在于、荒谬之处在于
- 真正……的是型、更……的是型

改法：删极值框架，直接给事实和判断。

### 拟人化否定动词

- 不认、不在乎、不理、不分、不挑、不接受
- 配身体类主语：血管、身体、细胞、器官
- 配数据类主语：数据、数字、账本、统计
- 配时空抽象类主语：历史、时间、市场、算法、制度

改法：铺具体机制，不用拟人化否定金句。

## 单字词

避免单字成句或连堆。

- 拆、拉、改、狠、空、冷、痛、停
- 单独成句的短词读起来生硬，要放进完整句子展开

## 用词偏好

- 冷静优先改为平静
- 不用接住、兜住
- 不用收敛、坍缩
- 不用两种都成立这个说法
- 减少很、偏
- 减少简短说结论、简单一句话这类提示

## 替换速查

| AI 味表达 | 更自然的处理 |
|-----------|--------------|
| 综上所述 | 所以，或直接说结论 |
| 鉴于此 | 因为这样 |
| 不难发现 | 删除，直接陈述 |
| 值得一提的是 | 删除，直接说 |
| 众所周知 | 删除，或补具体证据 |
| 具有重要意义 | 具体说对谁有什么用 |
| 这无疑是 | 删无疑，用事实支撑 |
| 在这个快速发展的时代 | 直接进具体问题 |
| 随着技术不断发展 | 说清具体变化是什么 |
| 让我们一起 | 改具体行动或直接收束 |
| 深入探索 | 直接说要分析什么，或删 |
| 复杂微妙 | 写清到底复杂在哪、分歧在哪 |
| 稳健 | 换具体能力、约束或测试结果 |
| 富有洞见 | 写出那条洞见本身 |
| 口径 | 说法、原话 |
| 定调 | 先表态、带头说 |
| 话术 | 说辞 |
| 通稿 | 统一文本 |
| 舆论场 | 几家媒体、几个平台 |
| 摊开来看 | 直接给判断 |

## 不机械替换

词表是默认，不是铁律。一个词在特定语境里是对的，就保留。遇到新口癖先判属于哪个问题族，按族处理，不急着加词。中英混排里的英文词按当前句子的实际语义判断，不机械套英文词表。
